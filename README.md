# ✈️ Indian Airlines Price Analysis Dashboard (Power BI)

## 📊 Project Overview
The **Indian Airlines Price Analysis Dashboard** is an interactive business intelligence project built using Power BI to analyze airline ticket price variations across different factors such as airline companies, travel routes, booking time, travel class, and flight duration.

The dashboard provides valuable insights into pricing trends within the Indian aviation market and helps understand how different variables influence ticket prices.

This project demonstrates how **data visualization and analytics can support pricing strategy analysis and travel planning decisions**.

---

# 🎯 Project Objectives

This dashboard aims to answer the following key business questions:

- Does ticket price vary across different airlines?
- How does the ticket price change when booking closer to the departure date?
- How do departure and arrival times affect ticket prices?
- How do ticket prices vary between different source and destination cities?
- What is the difference in pricing between **Economy and Business class**?
- How do **number of stops** impact ticket prices?

# 🖥️ Dashboard Pages

The report contains **two interactive dashboard pages**.

---

## 1️⃣ Main Dashboard – Airline Price Trends

This page focuses on high-level insights and pricing trends across airlines and travel timings.

### Key KPIs

- **Average Ticket Price:** ₹20.89K  
- **Maximum Ticket Price:** ₹123K  
- **Minimum Ticket Price:** ₹1K  
- **Total Flights Analyzed:** 300K  
- **Average Flight Duration:** 12.2 Hours  

### Visualizations

- **Price by Airline** – Comparison of average ticket prices across airlines.
- **Price by Days Left** – Analysis of how ticket prices increase as the departure date approaches.
- **Price by Departure Time** – Comparison of ticket prices across different departure time slots.
- **Price by Arrival Time** – Analysis of pricing trends based on arrival times.

These visuals help identify patterns in airline pricing behavior and booking timing.

## 2️⃣ Additional Analysis Dashboard

The additional analysis page provides deeper insights into flight pricing factors.

### Visualizations

- **Price by Route**  
  Displays average ticket price across major city routes.

- **Price by Class**  
  Comparison of pricing between **Economy and Business class** flights.

- **Price by Source City (Map Visualization)**  
  Geographic visualization of major Indian departure cities.

- **Price by Stops**  
  Shows how ticket prices vary between **direct flights and connecting flights**.

This page helps identify which routes, travel classes, and flight types drive price variations.

# 📂 Dataset Description

The dataset contains flight ticket pricing information across major Indian cities.

### Key Data Fields

| Column | Description |
|------|-------------|
| Airline | Airline operating the flight |
| Flight | Flight identifier |
| Source City | Departure city |
| Destination City | Arrival city |
| Departure Time | Flight departure time slot |
| Arrival Time | Flight arrival time slot |
| Stops | Number of stops in the flight |
| Class | Travel class (Economy or Business) |
| Duration | Total flight duration |
| Days Left | Days remaining before departure |
| Price | Ticket price |

---

# 📈 Key Insights

Some notable insights discovered from the dashboard:

- Ticket prices **increase significantly as departure dates approach**.
- Certain airlines consistently maintain **higher ticket pricing compared to competitors**.
- **Business class tickets are substantially more expensive** than economy class.
- **Non-stop flights generally cost more** compared to flights with stops.
- Ticket prices vary across different city routes, with certain routes showing higher average prices than others. Some routes such as **Chennai → Bangalore and Kolkata → Chennai** show slightly higher average ticket prices compared to other routes in the dataset.

# 🚀 How to Use the Dashboard

1. Open the Power BI report file.
2. Use the **filters on the left panel** to select:
   - Airline
   - Source City
   - Destination City
   - Class
   - Route Type
   - Departure Time
   - Arrival Time
3. Navigate between pages using the **dashboard navigation buttons**.
4. Interact with visuals to explore price patterns.
